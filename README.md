# Nazmus Sakib Raiyan — Portfolio

A simple portfolio website built with React (Vite) + Tailwind CSS.

## What’s Included
- Portfolio UI: Hero, About, Skills, Projects, Designs, Case Study
- Projects (folders under `projects/`):
  - `sos-system/` (README)
  - `chat-app/` (README)
  - `lead-generator/` (Node CLI)
  - `email-tool/` (Nodemailer script)
- Marketing case (markdown): `marketing/instagram-growth-strategy.md`
- Design samples: `public/designs/`

## Run Locally

1. Install dependencies:
	npm install
2. Start dev server:
	npm run dev
3. Build:
	npm run build
